## About Me

I build, test, automate, and improve digital products. My work combines software quality assurance, full-stack web development, cloud infrastructure, AI integrations, workflow automation, and practical solutions for real business problems.

I am the founder of **[Luxival](https://www.luxival.com)** — a Finnish technology and digital-services company based in Helsinki, Finland — delivering QA audits, web applications, automation systems, and premium digital experiences.

My approach is quality-first: I care about correctness, reliability, observability, and the user experience at every layer of the stack. I connect technical implementation directly to business requirements and client outcomes.

- Practical problem-solver who builds things that work in production
- Attention to detail at the code, test, architecture, and UX levels
- Continuous learner who builds working projects to consolidate new skills
- Engineering background (mechanical and industrial) that informs systems thinking
- Comfortable operating from requirements gathering through delivery and monitoring

### QA Suite Generator Agent

**Repository:** [oluwarotimitimileyin/qa-suite-generator](https://github.com/oluwarotimitimileyin/qa-suite-generator) &nbsp;|&nbsp; **Status:** Working

An agentic CLI tool powered by the Claude API. Given any target URL, the agent explores the website autonomously — navigating pages, identifying forms, buttons, and navigation elements — then generates a complete test suite to disk:

- Selenium + Python + pytest (Page Object Model)
- Playwright + Python + pytest
- BDD: behave + Gherkin feature files
- API tests: pytest + requests
- Cypress + JavaScript
- GitHub Actions CI configuration

Technologies: Python, Claude API (Anthropic), Selenium, Playwright, Cypress, pytest, behave
